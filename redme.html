Проект представляет собой трекер бюджета, написанный на Python с использованием библиотеки Tkinter для создания графического интерфейса. Он позволяет пользователям отслеживать свои доходы, расходы и цели по бюджетированию. Приложение имеет следующие функции: - Добавление доходов: пользователь может вводить суммы своих доходов, которые автоматически добавляются к общему бюджету. - Добавление расходов: пользователь может выбирать категорию расхода и вводить сумму расхода. Сумма расхода вычитается из общего бюджета. - Добавление целей: пользователь может добавлять финансовые цели, указывая их название и стоимость. - Отложение денег на цель: пользователь может выбирать цель и откладывать на неё определенные суммы, которые вычитаются из общего бюджета. - Просмотр целей: отображение списка добавленных пользователем целей с информацией о названии, стоимости, сумме, уже отложенной на цель и оставшейся сумме для отложения. - Просмотр расходов: отображение списка расходов по категориям с указанием сумм. Автором данного проекта является разработчик, имя которого не указано в предоставленном коде. Проект использует файл формата JSON для хранения данных о бюджете, целях и расходах. Весь функционал реализован с помощью объектно-ориентированного подхода с использованием классов и методов Tkinter для создания графического интерфейса.
